# Sample Packages

Ready-to-upload `.zip` packages for the AI Agent Marketplace — 4 skills and
3 agents. All are free, run offline (no API keys), and pass the upload
validator + security scanner.

Upload a `.zip` via the marketplace **+ Submit** page, then approve it in the
Admin review queue. The `src/` folder holds the unzipped source of each.

## Skills
- `text-summarizer.zip` — extractive text summarizer
- `word-counter.zip` — word/char/sentence counts + reading time
- `slugify.zip` — turn titles into URL slugs
- `json-prettifier.zip` — validate & pretty-print JSON

## Agents
- `email-drafter.zip` — bullet points -> polished email
- `csv-cleaner.zip` — trim/dedupe/normalize CSV
- `standup-generator.zip` — notes -> formatted daily standup
